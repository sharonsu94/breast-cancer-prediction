Feature Engineering

The purpose was to choose features with a high correlation with the target and a low correlation with other features

Feature Selection Methods

The different methods that we used were:

  • Filter Methods
    • Correlation Values
  • Wrapper Methods
    • Step backward
    • Exhaustive search
  • Embadded Methods
    • Lasso regulation
    • Random Forest recursive
    • Hybrid recursive

And then we tested all results with:

  • SVM (Support Vector Machine)
  • KNN (K Nearest Neighbor)
  • Random Forest
  • Naive Bayes
  • Deep Learning

Average Performance: C > R > H > L > E > B

Method Number of Features Performance
Correlation Values 20 ~98%
Random Forest recursive 10 95%~96%
Hybrid recursive 9 ~95%
Lasso regulation 12 93%~94%
Exhaustive search 4 92%
Step Backward 1 83%~88%

Based on the results, We discovered that filtering by Correlation Values created the best output

Correlation Values

We created this heatmap for the original dataset, there were some features had high correlations with each other, so we removed features that had a correlation value higher than 0.9 because of its duplicative nature

This code will filter out the features have high correlation with other features, we got the best combination when threshold = 0.9

This is the final heatmap for the correlation values. We only kept 20 of the 30 features

According to the table, we got the best outcome when we applyed SVM on Selected Features
